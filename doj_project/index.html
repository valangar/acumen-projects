Provider Profile Tool

Page updated: 24 May, 2019
Project: Ongoing - Acumen, LLC


This project is a large-scale software solution to connect multiple analyses for health care investigations.

PROCESS:

As one of the core developers of this web application, I am involved in designing and implementing visualizations, UI components and features in the tool.

TECH STACK:

  • JavaScript
  • D3.js for the interactive visualizations
  • Vue.js Framework
  • Webpack
  • Bootstrap
